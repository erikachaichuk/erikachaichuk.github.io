Real estate market analysis

This project utilizes data from Kaggle. The corresponding csv files can be found on Github.


Firsty, the properties data set was cleaned. The ID column was converted to string type to avoid unintentional numeric operations on the ID series. The column was also renamed to discard the irrelevant metadata.


Second, the customers data set was cleaned. There was an encoding issue with the ID column in the “properties”, and the first column in the “customers_raw_data” was also compromised. The column was thus renamed. Next, the entity column was turned from categoric to numeric by being renamed to ”individual” with values being mapped to 0 and 1. The “mortgage” variable was mapped to 0 and 1 as well.


Third, the two datasets were combined. That was possible through the “customer_ID” being the same feature in both tables. The initial merge resulted in missing some customer data because IDs in the “properties” table were flanked with empty spaces.


To overcome this problem, .strip() method was used, which removed the unnecessary spaces in both tables “customer_ID” series. The second attempt was successful, hence the joint table named “real estate data” was created. Before processing further, the data types of the combined table were verified.


The analysis phase began with calculating the age at purchase by taking it from the “birth_date” and “date_of_sale” fields.


Descriptive statistics and interval-based age groupings provided insights into the customers demographics. Property prices were similarly analyzed by grouping them into price intervals, getting the distribution of the sold and unsold properties.


To examine therelationship between age and property prices, covariance and correlation was utilized. The covariance revealed the extent of the relationship and whether the age impacts the price of properties. The correlation coefficient helped to explore the relationship further.


Geographical and categorical insights were explored through breakdowns. For example, the properties were analyzed by building to see the trends in average price, area, and deal satisfaction. A state-level breakdown included frequency, relative frequency, and cumulative frequency calculations.
